Assitive Technology For Vocally Impaired People

A device enabling vocally impaired people to communicate each other based on their laryngeal vibrations.

  • Laryngeal Signal Acquisition

    An electret condenser microphone based system is employed to pick up the laryngeal vibrations corresponding to each word

  • Feature Extraction

    The corresponding MFCC feature vectors of the laryngeal vibrations for each word is found out and stored in a database.

  • Feature Matching

    The feature matching is done to find the word subject intended to speak. During testing phase feature extraction of the testing word is done and it is compared with the features of the already existing words.

  • Sound Production

    The correct word identified using the feature matching process is then passed into the Microsoft SAPI which produces the sound.
